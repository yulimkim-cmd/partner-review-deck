# 파트너 미팅 프로세스

## 개요
파트너 미팅의 사전 준비부터 사후 정리까지 전체 프로세스.

---

## Phase 1. 사전 준비

### 1-1. 데이터 수집
- 파트너별 월별 송객·유입·전환율·확정률·취소율
- 상품(노선)별·도시별 실적
- 전년 동기 데이터 (YoY 비교용)
- 이전 미팅록·히스토리

### 1-2. 장표 제작

**장표 구조**

1. **기간별 파트너 실적 Overview** — 핵심 KPI(송객, 유입, 전환율) + 월별 YoY 테이블 + 성장/하락 분해
2. **도시별 파트너 실적** — 도시별 송객·유입·전환율 비교. 성장/하락 도시 식별
3. **노선별 파트너 실적·이슈 파악** — 상품(노선)별 송객순 정렬 + 주요 상품 하이라이트 + YoY
4. **제안 슬라이드 (1~2장, 선택)** — 미팅 목적에 따라 추가
5. **액션플랜** — 향후 협업 방향 + Next Action (담당·기한 명시)
6. **마무리**

**작성 규칙**

- 슬라이드 제목 = Action Title (제목이 곧 결론, 숫자 포함 필수)
- 파트너 대상 언어 사용 (아래 용어표 참조)
- stop-slop 적용
- 내부 데이터(GMV, 마진, 순위, 내부 전략) 제외

### 1-3. 내부 회의록 작성 (두괄식)

미팅 전 내부 공유용 문서. 장표와 별도로 작성.

```
1. 아젠다 (미팅에서 확인할 항목)
2. 장표 링크
3. 보조 지표 (장표에 넣지 않은 내부 데이터)
4. 팔로업 사항 (이전 미팅 액션 아이템 현황)
```

### 1-4. 아젠다 확정
- 내부 회의록의 아젠다를 기반으로 미팅 흐름 정리
- 파트너에게 공유할 내용 vs 내부용 확인 사항 구분

---

## Phase 2. 미팅

- 장표 기반으로 진행
- 파트너 피드백·요청 사항 기록
- 합의된 액션 아이템 실시간 정리

---

## Phase 3. 사후 정리

### 3-1. 미팅록 작성
- 요약 (2~3문장)
- 주요 논의 내용 (항목별 정리)
- Next Action 테이블 (담당 / 액션 / 기한)

### 3-2. 내부 공유
- 미팅록을 내부 회의록 하위 페이지로 등록
- 팔로업 액션 아이템 담당자 지정·공유

---

## 파트너 대상 언어

| 플랫폼 용어 | 파트너 용어 |
|------------|-----------|
| UV | 유입 / 유입량 / 방문자 |
| CVR | 전환율 |
| CTR | 클릭률 |
| imp | 검색 노출 |
| 퍼널 | 고객 여정 |
| MoM | 전월비 |

## 제외 항목 (파트너 장표에 넣지 않을 것)

- GMV / 거래액 / 마진 / 수수료
- 도시 TNA 내 순위
- 내부 아젠다 / 락인 화법 / 내부 전략
- Source 테이블명
- "실패" 등 부정적 단어 → 객관적 데이터로 대체
