# 파트너 리뷰 장표 생성

## 개요
파트너 대면 미팅용 실적 리뷰 장표를 생성하는 스킬.
MRT 디자인 시스템 + Action Title + stop-slop + 파트너 대상 언어로 작성.

## 장표 구조

1. **기간별 파트너 실적 Overview** — 리뷰 기간 전체 성과 요약. 핵심 KPI(송객, 유입, 전환율) + 월별 YoY 테이블 + 성장/하락 분해
2. **도시별 파트너 실적** — 파트너가 운영하는 도시별 송객·유입·전환율 비교. 성장/하락 도시 식별
3. **노선별 파트너 실적·이슈 파악** — 상품(노선)별 송객순 정렬 + 주요 상품 하이라이트 + YoY. 이슈가 있는 노선 식별
4. **제안 슬라이드 (1~2장, 선택)** — 미팅 목적에 따라 추가. 예시: 비수기 프로모션 제안, 고객 여정 딥다이브, 신규 상품 제안 등
5. **액션플랜** — 향후 협업 방향 + 구체적 Next Action (담당·기한 명시). 파트너 공유용 (내부 아젠다 제외)
6. **마무리**

## 디자인 원칙

### MRT 컬러
- MRT_BLUE: #00AFEC
- MRT_BLUE_DARK: #007BA3
- NAVY_DARK: #0D1B2A
- RED: #C0392B
- GREEN: #27AE60
- 폰트: Pretendard

### 헤더
MRT_BLUE_DARK 바(0.95") + RED 액센트 라인(4pt)

### 슬라이드 제목 = Action Title
- "매출 현황" ✗ → "송객 +60.8% 성장. 사파리 등 신규 4개 상품이 9,109명 창출" ✓
- 5초 안에 핵심 파악 가능해야 함
- 숫자 포함 필수

## 파트너 대상 언어 (필수)

| 플랫폼 용어 | 파트너 용어 |
|------------|-----------|
| UV | 유입 / 유입량 / 방문자 |
| CVR | 전환율 |
| CTR | 클릭률 |
| imp | 검색 노출 |
| 퍼널 | 고객 여정 |
| MoM | 전월비 |
| Source: MART_ | 마이리얼트립 내부 데이터 |
| KIND=1 확정 예약 | 확정 예약 기준 |
| 카테고리 기여도 | (삭제 또는 쉬운 표현) |

## 제외 항목 (파트너 장표에 넣지 않을 것)

- GMV / 거래액 / 마진 / 수수료
- 도시 TNA 내 순위 (#2, #12 등)
- 내부 아젠다 / 락인 화법 / 내부 전략
- Source 테이블명 (MART_SALE_D 등)
- "실패" 등 부정적 단어 → 객관적 데이터로 대체

## 데이터 소스

- 파트너별 월별 송객·유입·전환율·확정률·취소율 데이터
- 상품(노선)별·도시별 실적 데이터
- 전년 동기 데이터 (YoY 비교용)
- 이전 미팅록·히스토리 (내부 문서)

## 워크플로우

1. 파트너 확인 + 리뷰 기간 설정
2. 월별/상품별/도시별 실적 데이터 수집
3. YoY 비교 + 성장/하락 분해 분석
4. 장표 구조에 맞춰 슬라이드 작성
5. 파트너 대상 언어 교정
6. stop-slop 점검
